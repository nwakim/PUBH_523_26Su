Data transformations on columns

  • There are many functions that can be used to transform columns
  • We can use columns to create new variables, subset features, and reorganize a dataset
  • We will cover the most column functions:
    • mutate()
    • select()
    • rename()
    • relocate()

mutate()

mutate(): transform and create variables

  • mutate() allows us to transform existing columns and/or add new columns to the dataset

 

  • Think of this like adding a formula in Excel to calculate the value of a new column based on previous columns

 

  • mutate() is useful for:
    • Converting the units of one column to new units (e.g., months to years)
    • Performing arithmetic using multiple columns
    • Changing variable types or recoding categories

Example: calculating a new variable

  • Let’s convert age_months into years by dividing it by 12
  • We will name this new variable age_yrs
hrs_data |> 
  mutate(
1    age_yrs = age_months / 12
  )
1
Notice we use a single = inside mutate, not == or <-. This sets our new column name equal to the mathematical operation.

Example: calculating multiple columns and saving

  • We can create multiple new columns within a single mutate() function by separating them with a comma
  • Let’s save our work to a new object called hrs_data2 to avoid overwriting our original dataset
1hrs_data2 <- hrs_data |>
  mutate(
2    age_yrs = age_months / 12,
3    age_decades = age_yrs / 10
  )
1
We use the assignment operator <- to save the output into a brand new data frame called hrs_data2.
2
First, we create the age_yrs column.
3
Second, we create age_decades using the age_yrs variable we just defined in the step right above it!
  • Let’s take a look at the new data frame hrs_data2 to see our new columns

select()

Subsetting columns using select()

  • The select() function allows us to isolate specific variables or columns from our dataset
  • Why use select()?
    • Drops unneeded information to maximize processing speed and visual clarity
    • Isolates specific blocks of data for focused modeling or visualization

Example: selecting specific columns

  • Let’s keep only three variables from our dataset: age, smoke2now, and diab
    • Column names do NOT need quotes in select()
hrs_data2 |> 
  select(
1    age, smoke2now, diab
  )
1
We provide the exact column names we want to keep, separated by commas. All other columns will be dropped from the output.

Example: removing specific columns

  • If we want to keep everything except a few specified variables, we use a minus sign (-)
hrs_data2 |> 
  select(
1    -drink, -lung_dis
  )
1
Placing - in front of drink and lung_dis explicitly drops them, while retaining every other original variable.

Example: selecting consecutive columns

  • We can select a continuous block of columns using the : operator
hrs_data2 |> 
  select(
1    age:drink
  )
1
This keeps age, drink, and every single column positioned between them in the data frame. Make sure you know your column order before using this!

rename()

Renaming with rename()

  • You can rename columns with select() but it’s a bit easier to do this with the rename() function.

  • You just need to remember new_name = old_name ordering.

hrs_data3 <- hrs_data2 |> 
  rename(
    id = hhid,
    race = race_original,
    sex_bin = female
    )

relocate()

Reordering columns using relocate()

  • relocate() moves columns to a different horizontal position in your dataset
  • By default, variables sent into relocate() move directly to the very front of the dataset
1hrs_data3 |> relocate(lung_dis, arthritis)
2hrs_data3 |> relocate(lung_dis, .after = birthdate)
3hrs_data3 |> relocate(lung_dis, .before = race)
1
Moves lung_dis and arthritis to the very front of the dataset, before all other columns.
2
Moves income to place it directly behind the record column.
3
Moves income to place it right before the age column.

Wrap-up

  • We learned three functions that allow us to transform columns in our dataset:
    • mutate()
      • Transform existing columns and/or create new columns
    • select()
      • Subset specific columns to keep or drop
    • rename()
      • Rename columns with new names
    • relocate()
      • Reorder columns to a different spot in the dataset
