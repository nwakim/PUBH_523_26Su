Data visualization: Introduction to ggplot2

Nicky Wakim, Jessica Minnier, and Meike Niederhausen

Introduction to ggplot2

ggplot2 in tidyverse

  • ggplot2 is tidyverse’s data visualization package

    • This is one of the main ways to create plots and explore data

 

  • The gg in “ggplot2” stands for Grammar of Graphics

 

  • It is inspired by the book Grammar of Graphics by Leland Wilkinson

    • Make graphs/plots by combining independent components
    • Start with a basic plot then add layers

Works best with “tidy” data1

  1. Each variable must have its own column.

  2. Each observation must have its own row.

  3. Each value must have its own cell.

  1. Source: R for Data Science. Grolemund and Wickham.

Grammar of ggplot2

  • ggplot2 needs at least the following three to produce a chart:
    • data, a mapping, and a layer

 

  • For the most part, there are default settings for the other parts:
    • scales, facets, coordinates, and themes

Data

  • ggplot2 uses data to construct a plot

  • Works best with tidy data (when every observation is a row and each variable is a column)

  • First step in plotting:

    • Pass the data to the ggplot function, which stores the data to be used later by other parts of the plotting system

Data

  • For example, if we intend to make a graphic about the hrs_data dataset, we would start as follows:
ggplot(
  data = hrs_data
  )
1
We input the data that contains our variables of interest.

Mapping

  • Mappings use the aes() function to map variables to the different axes on a plot

    • aes() stands for “aesthetics”

Data + Mapping

  • If we want the age and income columns to map to the x- and y-coordinates in the plot, we can do that as follows:
ggplot(
  hrs_data, 
  mapping = aes(
    x = age,  
    y = income
    )
  )
1
We use the aes() function to specify the mapping of variables to axes. In this case, we are mapping age to the x-axis and income to the y-axis.

Layers

  • Every layer consists of three important parts:

    • The geometry that determines how data are displayed, such as points, lines, or rectangles

    • The statistical transformation that may compute new variables from the data and affect what of the data is displayed.

    • The position adjustment that primarily determines where a piece of data is being displayed

  • A layer can be constructed using the geom_*() and stat_*() functions

    • These functions often determine one of the three parts of a layer, while the other two can still be specified.

Data + Mapping + Layers

Here is how we can use two layers to display the age and income columns of the hrs_data dataset as points and stack a trend line on top:

ggplot(
  hrs_data, 
  mapping = aes(
    x = age, 
    y = income
    )
  ) + 
  geom_point() +
  geom_smooth(
    formula = y ~ x, 
    method = "lm"
    )
1
To create a scatterplot
2
To fit and overlay a line

ggplot(hrs_data) +

We can also make plots with a single variable

  • Data: still hrs_data

  • Mapping: using aesthetic to specify only one variable in the x-axis (age)

  • Layers: using geom_histogram() to show a plot of the counts per age

ggplot(
  hrs_data, 
  aes(age)
  ) +
  geom_histogram()
1
We specify only one variable in the aes() function, which will be mapped to the x-axis by default
2
We use geom_histogram() to create a histogram of the counts per `age

We can add more to plots!

We can change labels!

ggplot(
  hrs_data, 
  aes(age)
  ) +
  geom_histogram() +
  labs(
    x = "Age (years)",
    y = "Frequency",
    title = "Distribution of Age"
    )
1
We use the labs() function to change the labels of the axes and title of the plot
2
We change the x-axis label to “Age (years)”
3
We change the y-axis label to “Frequency”
4
We change the title of the plot to “Distribution of Age”

Adding more to plots!

Increase (or decrease) text size so we can read it / it fits nicely!

ggplot(
  hrs_data, 
  aes(age)
  ) +
  geom_histogram() +
  labs( 
    x = "Age (years)",
    y = "Frequency",
    title = "Distribution of Age"
    ) +
  theme(
    axis.text = element_text(size = 28),
    axis.title = element_text(size = 28),
    title = element_text(size = 28)
    )

Notice: common components of ggplot2

  • Common components of ggplot2 include:
    • ggplot(): the main function to create a plot
    • aes(): the function to specify the mapping of variables to axes
    • geom_*() or stat_*(): functions to add layers to the plot
    • labs(): the function to change labels of axes and title
    • theme(): the function to adjust non-data plot elements such as text size,

 

ggplot(
  data = my_data, 
  mapping = aes(x = col1, y = col2)
  ) +
  
  geom_point() +          
  
  labs() +       
  
  theme()

  • ggplot2 is great data visualization tool in R

    • We can add a lot of different components to our plots to make them more informative and visually appealing

 

  • We discussed some of the options in ggplot2 but there are many more to explore!
